!!! info "How Tags Work"

## Using Tags

- **Find related content** - Click any tag above to see all pages with that tag
- **Organize content** - Use consistent tag names across related pages
- **Discover content** - Browse tags to find documentation you might not know exists

## Adding Tags to Your Pages

Add tags in the front matter of your Markdown files:

```markdown
---
title: My Page Title
tags:
  - api
  - authentication
  - getting-started
categories:
  - Guides
---
```

See the [Using Tags Guide](guides/using-tags.md) for detailed instructions and best practices.
