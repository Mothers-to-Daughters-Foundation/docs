# Getting Started

Welcome! This guide will help you get started with using and contributing to our documentation.

## What is this documentation hub?

This is a static documentation site that:

- Stores all content as Markdown files in Git
- Automatically builds and deploys to GitHub Pages
- Provides search, navigation, and tagging features
- Tracks all changes through Git history

## For Readers

If you're here to read documentation:

1. Use the **navigation menu** on the left to browse topics
2. Use the **search bar** at the top to find specific content
3. Click on **tags** to find related content
4. Use the **"Edit this page"** link at the top if you spot an error

## For Contributors

If you want to contribute documentation:

1. Read the [Writing Documentation](../guides/writing-docs.md) guide
2. Learn how to [add new pages](../guides/adding-pages.md)
3. Understand [how to use tags](../guides/using-tags.md)
4. Submit your changes via a pull request
